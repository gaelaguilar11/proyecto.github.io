Ejercicios Prácticos de Java

Los siguientes ejercicios prácticos te ayudarán a poner en práctica tus conocimientos de Java. Cada ejercicio viene con una breve explicación y un código de ejemplo para que puedas comprender mejor los conceptos.

Ejercicios

Ejercicio 1: Suma de dos números

Realiza un programa en Java que solicite al usuario dos números enteros y muestre la suma de los mismos.


public class SumaNumeros {
    public static void main(String[] args) {
        int numero1 = 5;
        int numero2 = 7;
        int suma = numero1 + numero2;
        System.out.println("La suma de " + numero1 + " y " + numero2 + " es: " + suma);
    }
}

Ejercicio 2: Factorial de un número

Desarrolla un programa en Java que calcule el factorial de un número entero ingresado por el usuario.


public class Factorial {
    public static void main(String[] args) {
        int numero = 5;
        int factorial = 1;
        for (int i = 1; i <= numero; i++) {
            factorial *= i;
        }
        System.out.println("El factorial de " + numero + " es: " + factorial);
    }
}
